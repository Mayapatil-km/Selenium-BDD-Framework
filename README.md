# 🚀 Selenium BDD Automation Framework

## 📌 Project Overview
This project is a Selenium automation testing framework built using BDD approach.  
It is designed to test web applications using Cucumber, TestNG, and Page Object Model (POM).

---

## 🛠️ Technologies Used
- Java
- Selenium WebDriver
- Cucumber (BDD)
- TestNG
- Apache POI (Excel Data Driven Testing)
- Maven
- Extent Reports

---

## 📂 Project Structure
- Page Object Model (POM)
- Feature Files (Cucumber)
- Step Definitions
- Test Runner
- Utilities (Reusable methods)
- Hooks Class

---

## 📊 Features
- BDD Framework using Cucumber
- Page Object Model design pattern
- Data Driven Testing using Excel
- Reusable utility functions
- Screenshot on failure
- Extent Reports integration

---

## ▶️ How to Run Project
1. Clone the repository
2. Import project in Eclipse / IntelliJ
3. Install Maven dependencies
4. Run TestRunner class
5. Check execution reports in Report folder

---

## 📸 Reports
Extent Reports are generated after execution in the `/Report` folder.
